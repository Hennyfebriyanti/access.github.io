Pengenalan layout Responsif untuk halaman web - Layout adalah penetapan elemen halamn yang umumnya ada pada perancangan suatu system informasi maupun web. Layout responsif adalah salah satu layout yang sering diterpkan pada halaman website. Layout Responsif memungkinkan halaman web menyesuaikan diri dengan ukuran perangkat yang mengaksesnya, berikut ini adalah tampilan web yang sama ketika dibuka atau diakses menggunakan perangkat berbeda:.

Pentingkah layout responsive? Semakin hari semakin banyak orang yangmenggunakan perangkat mobile untuk browsing, seperti menggunakan smartphone dan tablet untuk setiap tugas atau pekerjaan sehai-hari. Pekerjaan yang dulnya hanya dilakukan melalui personal komputer (desktop & laptop) sekarang hampir semua sudah bisa digunakan juga melalui perangkat mobile.
